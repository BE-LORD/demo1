## 🎯 How It Works:

### Mouse Interaction:
- **Distance Detection**: Particles detect mouse position in 3D space
- **Repulsion Force**: Closer particles get pushed more
- **Smooth Return**: Particles smoothly return to original position
- **Real-time**: Updates every frame (60 FPS)

### Technical Details:
- Uses `useThree` hook for mouse pointer tracking
- Calculates distance between mouse and each particle
- Applies force based on distance
- Lerp (linear interpolation) for smooth movement
- Updates geometry buffer every frame
